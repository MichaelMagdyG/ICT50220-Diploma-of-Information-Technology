Security Policies
These security policies define the procedures, guidelines, and expectations to ensure the safety of company data, networks, and physical premises. All employees are required to adhere to the following security practices.

1. Data Protection

All sensitive data (employee, customer, financial) must be protected using encryption and secure access controls.
Any unauthorized disclosure of sensitive information is strictly prohibited.
2. Password Policy

Employees must use complex passwords that are a minimum of 12 characters and contain a mix of upper and lower case letters, numbers, and symbols.
Passwords must be changed every 90 days.
3. Access Control

Access to company systems, files, and networks is restricted based on job roles and responsibilities.
Only authorized users may access sensitive information or modify system settings.
4. Network Security

Firewalls, antivirus software, and intrusion detection systems must be implemented and regularly updated.
Remote access to the company network must be done through a Virtual Private Network (VPN).
5. Device Security

Employees must ensure that company-issued devices (laptops, smartphones, etc.) are password protected and encrypted.
Lost or stolen devices should be reported immediately to IT for remote wiping and secure deactivation.
6. Incident Response

Employees must immediately report any security breaches, suspicious activity, or incidents to the IT department.
An investigation will follow to assess the impact and implement corrective actions.
7. User Training

All employees will undergo regular security awareness training to stay informed about the latest threats and best practices for protecting company data.
